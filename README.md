# Functionality

We enter an expression into a string. Inside, the algorithm parses the expression into
Polish inverse notation and calculates the expression.

Example:  
  
calc   
enter expression (and press enter):  
1 + 2 * 3  
reverse polish notation is:  
1 2 3 * +  
result:  
7
